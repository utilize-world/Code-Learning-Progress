# State Machine


状态机就是用来管理状态的变换，要求每个状态之间相互独立，并且任何一个状态的改变都不会影响其他状态


## 创建状态机


说到底，根本思想在于，给每个状态都定义一个状态机，和当前状态的进入，更新和退出函数，所谓状态机，相当于链接各个状态的桥，而能不能过桥，什么时候过桥，并且到达状态后会发生什么，均存储在每个状态本身。而游戏中的各个实体，只需要调用当前状态自身的update，实际的各个行为模式都来自于各个状态
